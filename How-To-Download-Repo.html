Browser Source Overlays

This repository contains browser source overlays for use in streaming software such as OBS Studio or Streamlabs OBS.

Instructions for Downloading

Using GitHub Website

  1. Visit the Repository Page: Open your browser and go to KernFerm/browser-source-overlays.
  2. Download ZIP:
    • Click the green "Code" button near the top right of the repository page.
    • Select "Download ZIP" from the dropdown menu.
    • This will download a ZIP file containing the entire repository.
  3. Extract the ZIP File:
    • Locate the downloaded ZIP file on your computer.
    • Extract it using your operating system's built-in extraction tool or a third-party tool like WinRAR or 7-Zip.
